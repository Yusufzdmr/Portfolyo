## ✨ Özellikler

- 📱 Mobil uyumlu, responsive tasarım (Flexbox / Grid)
- ⚡ Hızlı yüklenen, sade ve modern arayüz

## 🛠️ Teknolojiler
`HTML5` · `CSS3` · `Vanilla JavaScript`

## 🚀 Çalıştırma
`index.html` dosyasını tarayıcıda açman yeterli — derleme gerekmez.

## 🌍 Canlı Yayın (GitHub Pages)
1. Repo → **Settings → Pages**
2. **Source:** `Deploy from a branch` → Branch: `main` → `/ (root)` → **Save**
3. Site birkaç dakika içinde yayında: `https://Yusufzdmr.github.io/Portfolyo/`

## 🏛️ Yapı
```
Portfolyo/
├── index.html      → ana sayfa
├── style.css       → stiller
├── assets/         → logo, manifest
└── images/         → görseller
```
